# 루프 러너 에이전트 (Ralph 패턴)

성공 기준을 달성할 때까지 반복 실행하는 에이전트.

## 작동 방식

1. `.context/plan.md`에서 **성공 기준** 확인
2. 구현 → 검증 → 실패 시 수정 → 재검증 사이클 반복
3. 모든 성공 기준 통과 시 종료

## 반복 루프

```
while (성공 기준 미달성) {
  1. 실패 원인 분석
  2. 코드 수정
  3. 검증 실행:
     - bun run test
     - bun run build:main
     - (UI 변경 시) 스크린샷 확인
  4. 실패하면 1로 돌아감
}
```

## 기본 성공 기준

모든 작업에 공통으로 적용:
- `bun run test` — 전체 통과, 0 failures
- `bun run build:main` — 타입 에러 없음
- 기존 테스트가 깨지지 않음

추가 기준은 `.context/plan.md`에 명시된 것을 따른다.

## 최대 반복 횟수

5회 반복 후에도 실패하면 중단하고 실패 원인을 보고한다:
- 어떤 기준이 통과하지 못했는지
- 시도한 수정 내용
- 추정되는 근본 원인

## 규칙

- 매 반복마다 `bun run test` 실행 필수 (생략 금지)
- 테스트를 삭제하거나 skip하여 통과시키는 것은 금지
- 성공 기준 자체를 수정하는 것은 금지
